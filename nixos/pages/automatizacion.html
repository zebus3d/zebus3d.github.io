⚙️ Automatización

¿Por qué automatizar? NixOS acumula espacio y generaciones con cada actualización. En lugar de limpiar manualmente, puedes configurar timers que lo hagan solos. Todo se configura en /etc/nixos/configuration.nix.

GC automático

Garbage Collector automáticoConfigura un timer que ejecuta GC automáticamente. Borra generaciones antiguas y libera espacio sin intervención manual.
Limpieza automática semanal.
● ● ● 
nix.gc = {
    automatic = true;
    dates = "weekly";
    options = "--delete-older-than 30d";
};
Optimización automáticaDeduplica archivos en el store periódicamente. No borra nada, solo ahorra espacio.
Deduplicar archivos.
● ● ● 
nix.settings.auto-optimise-store = true;

Entradas del bootloader

Limitar entradas del menú systemd-boot solo muestra las últimas N generaciones. Las entradas antiguas se eliminan automáticamente en cada rebuild.
Mantener el menú limpio.
● ● ● 
boot.loader.systemd-boot.configurationLimit = 3;

Esto le dice a systemd-boot que solo muestre las últimas 3 generaciones. Las entradas antiguas se eliminan automáticamente en cada nixos-rebuild.
