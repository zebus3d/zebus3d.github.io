📻 Radio en Tauon Music Box

Configuración de emisoras de radio españolas en Tauon Music Box (instalado vía NixOS). Tauon guarda las emisoras en un archivo llamado state.p dentro de ~/.local/share/TauonMusicBox/.

¿Qué es pickle? Pickle es una herramienta de Python para guardar datos en un archivo y recuperarlos después. Imagina que tienes una lista de emisoras escritas en un papel: pickle es como hacer una foto de ese papel para poder ver la misma lista más tarde. El archivo state.p es esa "foto" que contiene todas las emisoras que has añadido en Tauon. Si editas este archivo con un script de Python (como el de más abajo), puedes añadir o quitar emisoras sin tener que abrir Tauon.

Script para modificar state.p

Para añadir o modificar emisoras hay que editar el archivo state.p con un script de Python. El script abre el archivo, añade las emisoras y lo guarda.

Cómo funciona el script (explicado paso a paso):

  1. Montar módulos dummy (líneas 59-67): Como el script se ejecuta por separado (no dentro de Tauon), hay que "engañar" a Python para que piense que está dentro de Tauon. Esto se hace creando módulos falsos con types.ModuleType.
  2. Definir la clase RadioStation (líneas 69-76): Se copia la definición original de Tauon para que Python sepa qué forma tienen las emisoras (título, URL, país, etc.).
  3. Asignar la ruta de módulo correcta (líneas 78-80): Es como poner una etiqueta de dirección a la clase para que cuando Tauon lea el archivo, sepa que esta clase pertenece a Tauon y no a otro sitio.
  4. Abrir el archivo state.p (línea 84): Se abre en modo lectura binaria ("rb") porque pickle guarda los datos en un formato que solo entiende Python, no texto legible.
  5. Cargar el contenido (línea 85): pickle.load(f) lee el archivo y reconstruye los datos en una variable llamada save. Esta variable es una lista gigante donde cada número (142, 165) es un "cajón" que guarda información diferente.
  6. Añadir las emisoras (líneas 87-127): Se crean las emisoras y se meten en los cajones save[142] (lista principal) y save[165] (lista de reproducción).
  7. Guardar el archivo (líneas 129-132): Se escribe de vuelta el archivo en modo escritura binaria ("wb"). También se guarda una copia de seguridad (state.p.backup) para poder recuperarlo si algo sale mal.

¡IMPORTANTE! Tauon debe estar cerrado antes de ejecutar este script. Si no, Tauon sobrescribirá el archivo al cerrarse y perderás los cambios.

import pickle, os, sys, types
from dataclasses import dataclass

# Montar módulo dummy para que pickle resuelva la clase
tauon_mod = types.ModuleType('tauon')
sys.modules['tauon'] = tauon_mod
t_mod = types.ModuleType('tauon.t_modules')
sys.modules['tauon.t_modules'] = t_mod
tauon_mod.t_modules = t_mod
t_extra = types.ModuleType('tauon.t_modules.t_extra')
sys.modules['tauon.t_modules.t_extra'] = t_extra
t_mod.t_extra = t_extra

Notas importantes

La etiqueta de módulo es crítica: Al crear una RadioStation desde un script externo, hay que asignarle la etiqueta RadioStation.__module__ = 'tauon.t_modules.t_extra' antes de guardar. Si te olvidas, Python guarda la clase como si fuera de tu script y Tauon no la reconoce cuando intenta cargar state.p, perdiendo todas las emisoras que hayas añadido.

state.p se sobrescribe: Tauon guarda el estado al cerrar y periódicamente. Si los cambios no se cargaron correctamente (por el error del módulo), Tauon sobrescribe state.p con el estado vacío. Siempre hacer backup antes de modificar.

Geo-bloqueo: El stream oficial de Europa FM (atres-live.europafm.com) devuelve HTTP 451 fuera de España. Se usó un relay alternativo en Zeno FM. Onda Cero en cambio funciona sin geo-bloqueo.
