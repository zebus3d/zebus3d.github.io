📻 Radio en Tauon Music Box

Configuración de emisoras de radio españolas en Tauon Music Box (instalado vía NixOS). Tauon guarda las emisoras en state.p (pickle) dentro de ~/.local/share/TauonMusicBox/.

Estructura del estado

state.p — formato internoTauon serializa su estado con pickle. Las emisoras están en save[142] (prefs.radio_urls) y save[165] (radioplaylist_jar).

Script para modificar state.p

Para añadir/modificar emisoras hay que editar el pickle manteniendo la ruta de módulo correcta. Importante: RadioStation.__module__ debe ser 'tauon.t_modules.t_extra' o Tauon no podrá deserializarlo.

Script completoEjecutar con Tauon cerrado. Modifica state.p y state.p.backup.
Añade emisoras con URLs e iconos verificados.

import pickle, os, sys, types
from dataclasses import dataclass

# Montar módulo dummy para que pickle resuelva la clase
tauon_mod = types.ModuleType('tauon')
sys.modules['tauon'] = tauon_mod
t_mod = types.ModuleType('tauon.t_modules')
sys.modules['tauon.t_modules'] = t_mod
tauon_mod.t_modules = t_mod
t_extra = types.ModuleType('tauon.t_modules.t_extra')
sys.modules['tauon.t_modules.t_extra'] = t_extra
t_mod.t_extra = t_extra

Notas importantes

pickle y rutas de módulo: Al crear un RadioStation desde un script externo, hay que asignar RadioStation.__module__ = 'tauon.t_modules.t_extra' antes de picklear. Si no, pickle guarda la clase como __main__.RadioStation y Tauon lanza una excepción al cargar el state.p, perdiendo todos los cambios.

state.p se sobrescribe: Tauon guarda el estado al cerrar y periódicamente. Si los cambios no se cargaron correctamente (por el error del módulo), Tauon sobrescribe state.p con el estado vacío. Siempre hacer backup antes de modificar.

Geo-bloqueo: El stream oficial de Europa FM (atres-live.europafm.com) devuelve HTTP 451 fuera de España. Se usó un relay alternativo en Zeno FM. Onda Cero en cambio funciona sin geo-bloqueo.
