🧹 Garbage Collector

¿Qué es el Garbage Collector? En NixOS nada se borra automáticamente. Cada vez que actualizas el sistema, se crea una nueva generación (una foto completa de tu sistema) y todos los archivos se guardan en /nix/store. Con el tiempo esto acumula gigas de versiones que ya no usas.

El GC es el comando que libera ese espacio. Primero borras generaciones (los "punteros") y luego el GC elimina los archivos reales que ya nadie referencia. Es como vaciar la papelera de reciclaje.

La generación actual (la que estás usando ahora) NUNCA se elimina. Siempre puedes hacer rollback.

IMPORTANTE: No tener compilaciones o descargas activasAplica a: nix-collect-garbage, -d y --delete-older-than. Si nixos-rebuild está descargando/compilando y el GC borra esas referencias, falla con "path does not exist". al ejecutar el GC.

RECOMENDADO
GC conservador (30 días)Borra generaciones con +30 días de TODOS los perfiles y ejecuta GC. Mantiene la generación actual y las de los últimos 30 días. Ideal para uso semanal sin perder rollback.Ej: rebuild de hace 35 días ▶ se borra
Generaciones viejas + espacio liberado en un paso.
sudo nix-collect-garbage --delete-older-than 30d
MODERADO
GC simpleSolo elimina archivos del store sin referencias. No borra generaciones. Útil cuando ya borraste generaciones a mano y quieres liberar el espacio. ⚠️ No actualiza el menú de systemd-boot: ejecuta nixos-rebuild boot después para sincronizar las entradas.Ej: tras borrar generaciones manualmente, libera GB
Libera espacio sin borrar generaciones. Requiere nixos-rebuild boot después para actualizar el bootloader.
sudo nix-collect-garbage
AGRESIVO
GC agresivo (-d)Borra TODAS las generaciones excepto la actual. Úsalo con precaución si necesitas espacio. ⚠️ Elimina TODOS los rollbacks.Ej: solo queda la generación actual
Borra todas las generaciones anteriores.
sudo nix-collect-garbage -d
SEGURO
Optimizar storeDeduplica archivos iguales en el store. No borra nada, solo organiza para ahorrar espacio.Ej: misma librería en 50 programas ▶ 1 copia
Deduplica archivos en el store.
sudo nix-store --optimise
