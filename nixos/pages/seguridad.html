🛡️ Seguridad

Vulnerabilidades del kernel y cómo mitigarlas en NixOS. Mantener el sistema blindado es tan importante como mantenerlo actualizado.

CVE-2026-31431 — "Copy Fail"

¿Qué es? Vulnerabilidad de escalada de privilegios (CVSS 7.8) en el kernel Linux entre versiones 4.14 y 6.19.12. Permite a un usuario local escalar a root explotando el módulo algif_aead (interfaz de criptografía por userspace vía AF_ALG).

Más información: copy.fail · NVD

MitigaciónBloquea el módulo vulnerable para que no se cargue nunca.
Blacklist del módulo en la configuración del sistema.
● ● ● nix 
# En /etc/nixos/configuration.nix:
boot.blacklistedKernelModules = [ "algif_aead" ];

Luego reconstruir: sudo nixos-rebuild switch --flake /etc/nixos

Verificar que está activoComprueba que el módulo no está cargado y que el blacklist funciona.
Comandos de verificación.
● ● ● bash 
# Verificar que el módulo NO está cargado:
lsmod | grep algif

# Verificar el blacklist en la configuración:
grep algif_aead /etc/nixos/configuration.nix

# Verificar que modprobe lo tiene blacklisted:
cat /etc/modprobe.d/nixos.conf | grep algif_aead

¿Por qué funciona? boot.blacklistedKernelModules en NixOS genera un archivo en /etc/modprobe.d/nixos.conf con la directiva blacklist algif_aead. Esto impide que el módulo se cargue automáticamente durante el arranque o por demanda vía alias. El módulo solo podría cargarse manualmente con modprobe algif_aead (sin -b), lo cual requiere privilegios de root — y si un atacante ya tiene root, el daño está hecho.

¿Cuándo quitar esta mitigación? Cuando el kernel se actualice a una versión superior a 6.19.12, puedes eliminar esta línea. Verifica siempre con uname -r que el kernel ya no sea vulnerable.

Otros recursos

Además del blacklist de módulos, mantén el sistema actualizado y revisa advisories de seguridad del kernel regularmente. En NixOS, actualizar el canal (nixos-rebuild switch --upgrade) suele ser suficiente para la mayoría de vulnerabilidades del kernel, pero algunas (como Copy Fail) requieren mitigación manual hasta que llegue el parche.
