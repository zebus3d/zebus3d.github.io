📦 Instalar y buscar paquetes

¿Cómo se instala software en NixOS? Hay dos formas: declarativa (añades el paquete a configuration.nix y se instala al hacer rebuild) o imperativa (lo instalas al momento con nix-env). La declarativa es la recomendada porque es reproducible y persistente.

📜 La carta a los Reyes Magos — La configuración declarativa es como escribir la carta a los Reyes Magos: escribes todos tus deseos en un papel (el archivo configuration.nix), describes exactamente qué quieres tener en tu sistema, y cuando ejecutas nixos-rebuild switch "llegan los Reyes" y te traen todos los regalos de golpe. Si quieres algo nuevo, lo añades a la carta y listo. Así de fácil: escribes lo que quieres y ¡voilà!, ya lo tienes.

¿Dónde está mi configuración? El archivo principal es /etc/nixos/configuration.nix. Ahí se define TODO: paquetes, servicios, usuarios, red, etc.

Buscar paquetes

Buscar en nixpkgsBusca paquetes en el repositorio oficial. Requiereexperimental-features configurado.
Buscar paquetes por nombre.
● ● ● bash 
nix search nixpkgs firefox

Requiere tener en configuration.nix: nix.settings.experimental-features = [ "nix-command" "flakes" ];

Instalar (declarativo — recomendado)

Instalar paquetes del sistemaAñade paquetes a environment.systemPackages. Se instalan al hacer rebuild y persisten.
El método recomendado.
● ● ● bash 
environment.systemPackages = with pkgs; [
    firefox
    git
    vlc
];

Luego reconstruye:

● ● ● bash 
sudo nixos-rebuild switch

Instalar (imperativo — rápido)

Instalar para el usuario actualInstala el paquete solo para el usuario actual. No requiere sudo, no persiste tras reboot.
Instala temporalmente.
● ● ● bash 
nix-env -iA nixpkgs.firefox
Entorno temporal (nix-shell)Crea un entorno con paquetes sin instalarlos permanentemente. Cierra la shell y desaparecen.
Ambiente temporal.
● ● ● bash 
nix-shell -p python3 --run "python --version"

Rollback — Deshacer cambios

Volver a la generación anteriorDeshace el último rebuild. Si algo no funciona, esto te salva.
Si algo falla, esto lo arregla.
● ● ● bash 
sudo nixos-rebuild switch --rollback
