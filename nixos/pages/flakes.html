❄️ Flakes

¿Qué son los Flakes? Son la forma moderna de gestionar tu configuración de NixOS. Imagina que tu configuración es un "proyecto" con sus propias dependencias versionadas, como un proyecto de programación con su package.json.

El archivo flake.nix declara qué usar y el flake.lock "congela" las versiones exactas para que la compilación sea siempre idéntica. Esto hace que tu sistema sea reproducible: si copias los archivos a otro ordenador, obtienes exactamente lo mismo.

Canales vs Flakes: Los canales son como un "apt update" global (simple pero menos control). Los flakes son proyectos independientes con dependencias versionadas (más control, más reproducible).

⚠️ Requiere configuración

Para usar flakes necesitas añadir esta línea en tu configuration.nix:

nix.settings.experimental-features = [ "nix-command" "flakes" ];

Sin esto, los comandos de flakes no funcionarán.

Verificar tipo de configuración

¿Canales o Flakes?Cómo saber qué tipo de configuración estás usando.
Ejecuta este comando para saber qué sistema usas.
● ● ● bash 
nix-channel --list
Si muestra nixos https://nixos.org/channels/... ▶ Canales. Si no ▶ Flakes.

Estructura típica

Estructura de proyecto con FlakesOrganización de un proyecto con flakes. flake.nix declara qué dependencias usar (nixpkgs, home-manager). flake.lock guarda las versiones exactas (como package-lock.json). nixos/ contiene la configuración del sistema.├── flake.nix ▶ define inputs y outputs ├── flake.lock ▶ versiones exactas └── nixos/ ▶ config del sistema
Cómo organizar tu configuración con Flakes.
● ● ● bash 
mi-config/
├── flake.nix        # Dependencias y outputs
├── flake.lock       # Versiones congeladas
└── nixos/
    ├── configuration.nix
    └── hardware-configuration.nix

Comandos útiles

Actualizar FlakeActualiza todas las dependencias a sus últimas versiones según flake.nix.
Actualizar todas las dependencias.
● ● ● bash 
nix flake update
Actualizar un input específicoActualizar solo una dependencia específica (ej: nixpkgs-unstable) sin tocar las otras.
Actualizar solo una dependencia.
● ● ● bash 
nix flake lock --update-input nixpkgs-unstable
Evaluar configuraciónVer la configuración sin aplicar cambios. Útil para depurar.
Probar la configuración.
● ● ● bash 
sudo nixos-rebuild dry-build --flake .#nixos
