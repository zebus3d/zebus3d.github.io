🗑️ Generaciones

¿Qué son las generaciones? Cada vez que haces nixos-rebuild switch, NixOS guarda una "foto" completa de tu sistema. Estas fotos se llaman generaciones.

Si algo sale mal después de un cambio, puedes volver atrás (rollback) a cualquier generación anterior. Es como tener un control de versiones del sistema entero.

La generación actual NUNCA se borra. Siempre puedes volver a ella. Las generaciones son perfiles en /nix/var/nix/profiles/system/

Manejar generaciones

Ver generacionesLista todas las generaciones con su fecha, versión de NixOS y si son la actual.
Ver qué generaciones tienes disponibles.
● ● ● bash 
sudo nix-env -p /nix/var/nix/profiles/system --list-generations
Borrar generaciones específicasBorra generaciones por número. Útil para limpiar generaciones antiguas.sudo nixos-rebuild delete-generations 1 2 3 4 5
Eliminar generaciones antiguas manualmente.
● ● ● bash 
sudo nixos-rebuild delete-generations 1 2 3
RollbackVolver a la generación anterior. Es el método más rápido para revertir cambios.
Volver al estado anterior.
● ● ● bash 
sudo nixos-rebuild switch --rollback
Elegir generación específicaSeleccionar una generación exacta por número.Útil si sabes cuál funcionaba bien.sudo nixos-rebuild switch --profile 5
Cambiar a una generación específica.
● ● ● bash 
sudo nixos-rebuild switch --generation <número>

Limpiar generaciones huérfanas

Consejo: Tras borrar generaciones, puedes liberar el espacio ejecutando:

● ● ● bash 
sudo nix-collect-garbage
